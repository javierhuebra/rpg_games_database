Aplicación web para registrar resultados de juego de mesa orientada a dispositivos móviles por su portabilidad.
Consiste en un tablero de jugadores dinámicos y generación de turnos aleatorios para ordenar el juego sin perder tiempo.
Es posible registrar los resultados del final de la partida mediante un código único por grupo de juego para generar un historial de partidas y acceder al mismo con el código.
